About Easy Post

Easy Post is a dynamic, flask template for handling POST / GET request.

Let's say you want to make a login form, suvey, voting poll or any kind of form for collecting data while being able to store the collected data in formats like json, csv, plain text and even xml while seamlessy hosting the (form / polls) in either a local web-server or in the internet.

EasyPost makes doing all that super simple, with it's GUI interface and fully opensource templates. It's lightweight fast, stable* And the best part the data never leaves your device.
